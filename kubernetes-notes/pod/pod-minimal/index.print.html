Definición pod mediante manifiesto

Manifiesto mínimo

[dzamo@victus my-codes]$ cat kubernetes/01-pod-minimal.yaml
apiVersion: v1
kind: Pod
metadata:
  name: nginx
spec:
  containers:
  - name: nginx
    image: nginx:alpine

Aplicando manifiesto (crear pod)

[dzamo@victus my-codes]$ kubectl apply -f kubernetes/01-pod-minimal.yaml
