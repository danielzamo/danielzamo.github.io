Jekyll en Fedora 40

Instalación de pre requisitos

sudo dnf -y install ruby ruby-devel openssl-devel \
 redhat-rpm-config gcc-c++ @development-tools

Verificaciones

ruby -v; gcc -v; g++ -v; make -v

Instalando RubyGems

Evitar instalar los paquetes de RubyGems (llamados gemas) como usuario root. En su lugar, configurar un directorio de instalación de gemas para una cuenta de usuario. Los siguientes comandos agregarán variables de entorno al ~/.bashrc para configurar la ruta de instalación de gemas.

echo '# Install Ruby Gems to ~/.config/gems' >> ~/.bashrc
echo 'export GEM_HOME="$HOME/.config/gems"' >> ~/.bashrc
echo 'export PATH="$HOME/.config/gems/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc

Instalar Jekyll y Bundler

gem install jekyll bundler

Otros comandos

Instalar/update RubyGems

gem update --system

Jekyll - guía de inicio rápida

Inicio rápido

Una vez instalado el generador de sitios Jenkyll se puede crear un sitio web estático. Luego este se podra modificar tanto la apariencia, las URL, los datos que se muestran en la página y más.

La instalación del generador de sitios estático (SSG) y sus dependencias se pueden consultar en Jekyll y/o en este mismo sitio.

Crear un nuevo sitio Jekyll

jekyll new mysite

cd mysite

Contruir el sitio y desplegarlo en local

bundle exec jekyll serve

Si todo a ido bien el sitio debería estar disponible en el URL http://localhost:4000.
